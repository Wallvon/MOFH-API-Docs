# Remove account
This endpoint removes the vPanel user account associated with the provided information.


## HTTP Endpoint

`POST https://panel.myownfreehost.net/json-api/removeacct.php`

## Usage

<Callout emoji="âš ï¸" type="warning">
  You will first have to use the [suspend](/JSON/accounts/suspend) endpoint to suspend the account before you can remove it.
</Callout>

<Tabs items={['cURL', 'Python']}>
    <Tab>
        ```bash
        curl -X POST -u username:password -d "user=example" "https://panel.myownfreehost.net/json-api/removeacct.php"
        ```
    </Tab>
    <Tab>
        ```python
        import requests
        from requests.auth import HTTPBasicAuth

        url = "https://panel.myownfreehost.net/json-api/removeacct.php"
        data = {'user': 'username'} # For the username, use the same username you set when creating the account.

        response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

        print(response.json())
        # Do this if you only want to get the status code.
        print(response.json()["result"][0]["status"])
        ```
    </Tab>
</Tabs>
