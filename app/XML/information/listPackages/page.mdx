# List packages

This endpoint lists the packages available for the reseller.

<Callout type="error" emoji="❗️">
  This API endpoint is <u>**broken**</u>. Use the [JSON
  endpoint](/JSON/information/list-packages) instead.
</Callout>

## HTTP Endpoint

`GET https://panel.myownfreehost.net/xml-api/listpkgs.php`

## Usage

<Tabs items={['cURL', 'Python']}>
    <Tabs.Tab>
        ```bash
        curl -u username:password "https://panel.myownfreehost.net/xml-api/listpkgs.php"
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```python
        import requests
        from requests.auth import HTTPBasicAuth

        url = "https://panel.myownfreehost.net/xml-api/listpkgs.php"

        response = requests.get(url, auth=HTTPBasicAuth('username', 'password'))

        print(response.text)
        ```
    </Tabs.Tab>

</Tabs>
