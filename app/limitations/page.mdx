# Limitations

## API limitations

The MyOwnFreeHost API is very limited. Whilst you can <i>theoretically</i> do more than what is listed here, and many hosting providers have, this cannot be done using the API. You would have to make calls to VPanel directly. As of right now, we don't have any documentation on how to do this.

## Account limit

<Callout type="error">
  You are allowed to create up to **3** accounts per person with the
  MyOwnFreeHost API.
</Callout>

Whilst you _may_ be able to create more accounts, this is not advised. When contacted, the iFastNet team also said that there was a limit of 3 accounts per person. **If you do create more, please be wary of the risks.**

## API responses

**Every** request returns a <GreenCallout>200 OK</GreenCallout> status code, even invalid requests. You
need to check the response body to see if the request was successful or not.

## API SSL certificate

<Callout type="warning">
  The SSL certificate on `https://panel.myownfreehost.net/xml-api/` tends to
  expire, so you may need to turn off SSL verification.
</Callout>
