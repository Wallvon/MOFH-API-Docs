# Authentication

<Callout type="error">
  <u>**All**</u> API requests, except for requests to
  [version.php](/information/apiVersion), must be authenticated.
</Callout>

<Callout type="important">
  You must replace `username` and `password` with your API credentials.
</Callout>

MyOwnFreeHost uses API credentials to allow access to the API. You can get them [here](https://panel.myownfreehost.net/panel/index2.php?option=api).

## Usage

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
            ```bash copy showLineNumbers filename="Shell"
            curl -X POST -u username:password "https://panel.myownfreehost.net/json-api/"
            ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth

            url = "https://panel.myownfreehost.net/json-api/" # This is the base URL
            data = {}

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))
            ```
      </Tabs.Tab>

  </Tabs>
</JsonOnly>
