# Unsuspend account

This endpoint unsuspends the vPanel user account associated with the provided information.

## HTTP Endpoint

`POST https://panel.myownfreehost.net/json-api/unsuspendacct.php`

## Usage

<Tabs items={['cURL', 'Python']}>
    <Tabs.Tab>
        ```bash
        curl -X POST -u username:password -d "user=username" "https://panel.myownfreehost.net/json-api/unsuspendacct.php"
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        ```python
        import requests
        from requests.auth import HTTPBasicAuth

        url = "https://panel.myownfreehost.net/json-api/unsuspendacct.php"
        data = {'user': 'username'} # For the username, use the same username you set when creating the account.

        response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

        print(response.json())

        # Do this if you only want to get the status code.
        print(response.json()["result"][0]["status"])
        ```
    </Tabs.Tab>

</Tabs>

## Response

<Callout emoji="💡">
    **The above request returns JSON structured like the following:**

    ```json
    {
        "result": [
            {
                "status": 1,
                "statusmsg": "The account username is now unsuspending"
            }
        ]
    }
    ```

</Callout>
