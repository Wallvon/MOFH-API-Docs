# Get API version

<Callout type="important">
  This API call does not require authentication.
</Callout>

This endpoint gets the current API version.

## HTTP Endpoint

<br />
<JsonOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/json-api/version.php
    ```
  </code>
</JsonOnly>
<XmlOnly>
  {/* prettier-ignore */}
  <code>

  </code>
</XmlOnly>
