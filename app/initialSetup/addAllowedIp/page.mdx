# Add an allowed IP

<Callout type="important">
  The MyOwnFreeHost API requires you to add the IP address of the server you are
  going to make API requests from.
</Callout>

## Setup

<Steps>
### Log in to the MyOwnFreeHost reseller panel

The reseller panel can be accessed at [https://panel.myownfreehost.net/index.php](https://panel.myownfreehost.net/index.php). The page should look something like this:

![Login page](/images/login.png)

When you log in, the dashboard looks something like this:

![Dashboard](/images/dashboard.png)

### Go to the API setup page

This can be done by clicking on the `API` link in the sidebar, and then clicking on `Setup WHM API`. Once you have done that, this is what the page should look like:

![WHM API page](/images/whm-api.png)

Then, select the domain name you want to use and click on `Get Keys/ Set allow ip address`.

### Add the IP address

Once you are there, add your IP address to the `API Allowed IP Address (main IP of server you are connecting from)` field. It should look something like this:

![IP address added](/images/api-ip.png)

<Callout type="info">In this example the IP `1.1.1.1` is used.</Callout>

Then, click on `Update Settings` and everything should be set up!

</Steps>
