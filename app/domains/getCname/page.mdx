# Validate domain by CNAME

<Callout type="error">
  You <u>**cannot**</u> create new accounts using a CNAME verified domain name.
  You can only add a CNAME verified domain name to an existing account.{" "}
  <u>**This has to be done manually in vPanel.**</u>
</Callout>

This endpoint allows you to add a domain using CNAME validation instead of DNS validation.

It uses a <YellowCallout>POST</YellowCallout> request, which is confusing because it's called `getcname`.

After using this endpoint, use the hash it returns and add that as a subdomain to your domain. Point this subdomain to `ns1.byet.org`.

## Example

Let's say your domain is `example.com`. You would make a request with `example.com` as value for the `domain` parameter.

This returns a hash. In this case we'll use `73081144a0525fde6ba1b0510684efcf`.

The user would create a subdomain called `73081144a0525fde6ba1b0510684efcf.example.com` and point it to `ns1.byet.org`.

Afterwards, the user can add the domain name to vPanel.

## Usage

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "api_user=username&api_key=password&domain_name=subdomain.example.com" "https://panel.myownfreehost.net/json-api/getcname.php"
          ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth

            url = "https://panel.myownfreehost.net/json-api/getcname.php"
            data = {'api_user': 'username', 'api_key': 'password', 'domain_name': 'subdomain.example.com'}

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.text)
            ```
      </Tabs.Tab>
