# Check if a domain is available

This endpoint checks if the requested domain name is available.

## HTTP Endpoint

<br />
<JsonOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/json-api/checkavailable.php
    ```
  </code>
</JsonOnly>
<XmlOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/xml-api/checkavailable.php
    ```
  </code>
</XmlOnly>

## Parameters

<Callout type="important">
  The API username and key must be provided in both the parameters and basic
  auth.
</Callout>

<Callout type="important">`domain` can also be a subdomain.</Callout>

## Usage

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "api_user=username&api_key=password&domain=subdomain.example.com" "https://panel.myownfreehost.net/json-api/checkavailable.php"
          ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth

            url = "https://panel.myownfreehost.net/json-api/checkavailable.php"
            data = {'api_user': 'username', 'api_key': 'password', 'domain': 'subdomain.example.com'} # If you want to use a domain which is not a subdomain, put that domain in.

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.json())
            ```
      </Tabs.Tab>

  </Tabs>
</JsonOnly>
