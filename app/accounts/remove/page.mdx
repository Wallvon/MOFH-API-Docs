# Remove account

<Callout type="warning">
  You will first have to [suspend the account](/accounts/suspend) before you can
  remove it.
</Callout>

This endpoint removes the vPanel user account associated with the provided information.

## HTTP Endpoint

<br />
<JsonOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/json-api/removeacct.php
    ```
  </code>
</JsonOnly>
<XmlOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/xml-api/removeacct.php
    ```
  </code>
</XmlOnly>

## Parameters

<Callout type="important">
  `user` is the username you provided when creating the user account using
  [createacct.php](/accounts/create). This is <u>**not**</u> the vPanel username
  (hname_12345678).
</Callout>

## Usage

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "user=username" "https://panel.myownfreehost.net/json-api/removeacct.php"
          ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth

            url = "https://panel.myownfreehost.net/json-api/removeacct.php"
            data = {'user': 'username'} # For the username, use the same username you set when creating the account.

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.json())
            # Do this if you only want to get the status code.
            print(response.json()["result"][0]["status"])
            ```
      </Tabs.Tab>

  </Tabs>
</JsonOnly>
<XmlOnly>
  <Tabs items={['cURL', 'Python']}>
    <Tabs.Tab>
            ```bash copy showLineNumbers filename="Shell"
            curl -X POST -u username:password -d "user=username" "https://panel.myownfreehost.net/xml-api/removeacct.php"
            ```
    </Tabs.Tab>
    <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth
            from xml.etree.ElementTree import fromstring, ElementTree # Do this if you only want to get the status code.

            url = "https://panel.myownfreehost.net/xml-api/removeacct.php"
            data = {'user': 'username'} # For the username, use the same username you set when creating the account.

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.text)

            # Do this if you only want to get the status code.
            tree = ElementTree(fromstring(response.content))
            root = tree.getroot()
            for child in root:
                print(child[0].text) # The status code (can either be 0 or 1)
            ```
    </Tabs.Tab>

  </Tabs>
</XmlOnly>

## Response

<JsonOnly>
    ```json copy showLineNumbers filename="Response"
    {
        "result": [
            {
                "statusmsg": "username account removed",
                "rawout": "username queued for removal",
                "status": 1
            }
        ]
    }
    ```

</JsonOnly>
<XmlOnly>
    ```xml copy filename="Response"
    <removeacct>

            <statusmsg>

                username account has been unsuspended
            </statusmsg>
        </result>
    </removeacct>
    ```
</XmlOnly>
