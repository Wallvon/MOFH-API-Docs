# Change password

This endpoint changes the password of the vPanel user account associated with the provided information.

## HTTP Endpoint

<br />
<JsonOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/json-api/passwd.php
    ```
  </code>
</JsonOnly>
<XmlOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/xml-api/passwd.php
    ```
  </code>
</XmlOnly>

## Parameters

<Callout type="important">
  `user` is the username you provided when creating the user account using
  [createacct.php](/accounts/create). This is <u>**not**</u> the vPanel username
  (hname_12345678).
</Callout>

## Usage

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "user=username&pass=password" "https://panel.myownfreehost.net/json-api/passwd.php"
          ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth

            url = "https://panel.myownfreehost.net/json-api/passwd.php"
            data = {'user': 'username', 'pass': 'password'}  # For the username, use the same username you set when creating the account.

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.json())

            # Do this if you only want to get the status code.
            print(response.json()["passwd"][0]["status"])
            ```
      </Tabs.Tab>

  </Tabs>
</JsonOnly>
<XmlOnly>
  <Tabs items={['cURL', 'Python']}>
    <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "user=username&pass=password" "https://panel.myownfreehost.net/xml-api/passwd.php"
          ```
    </Tabs.Tab>
    <Tabs.Tab>
          ```python copy showLineNumbers filename="Python"
          import requests
          from requests.auth import HTTPBasicAuth
          from xml.etree.ElementTree import fromstring, ElementTree # Do this if you only want to get the status code.

          url = "https://panel.myownfreehost.net/xml-api/passwd.php"
          data = {'user': 'username', 'pass': 'password'}  # For the username, use the same username you set when creating the account.

          response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

          print(response.text)

          # Do this if you only want to get the status code.
          tree = ElementTree(fromstring(response.content))
          root = tree.getroot()
          for child in root:
              print(child[5].text) # The status code (can either be 0 or 1)
          ```
    </Tabs.Tab>

  </Tabs>
</XmlOnly>
