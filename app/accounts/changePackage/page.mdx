# Change package

This endpoint changes the package of the vPanel user account associated with provided information.

## HTTP Endpoint

<br />
<JsonOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/json-api/changepackage.php
    ```
  </code>
</JsonOnly>
<XmlOnly>
  {/* prettier-ignore */}
  <code>
    ```console copy filename="URL"
    https://panel.myownfreehost.net/xml-api/changepackage.php
    ```
  </code>
</XmlOnly>

## Parameters

<Callout type="important">
  `user` is the username you provided when creating the user account using
  [createacct.php](/accounts/create). This is <u>**not**</u> the vPanel username
  (hname_12345678).
</Callout>

## Usage

<Callout type="important">
  Your package name <u>**must**</u> be fully lowercase since otherwise the user
  will get the `Client is not allocated to a package` package which has the
  maximum resources.
</Callout>

<JsonOnly>
  <Tabs items={['cURL', 'Python']}>
      <Tabs.Tab>
          ```bash copy showLineNumbers filename="Shell"
          curl -X POST -u username:password -d "user=example&pkg=myawesomeplan" "https://panel.myownfreehost.net/json-api/changepackage.php"
          ```
      </Tabs.Tab>
      <Tabs.Tab>
            ```python copy showLineNumbers filename="Python"
            import requests
            from requests.auth import HTTPBasicAuth
            # The JSON API endpoint also returns XML
            from xml.etree.ElementTree import fromstring, ElementTree # Do this if you only want to get the status code.

            url = "https://panel.myownfreehost.net/json-api/changepackage.php"
            data = {'user': 'username', 'pkg': 'myawesomeplan'} # Keep in mind that the package name must be fully lowercase.

            response = requests.post(url, params=data, auth=HTTPBasicAuth('username', 'password'))

            print(response.text)

            # Do this if you only want to get the status code.
            tree = ElementTree(fromstring(response.content))
            root = tree.getroot()
            for child in root:
                print(child[1].text) # The status code (can either be 0 or 1)
            ```
      </Tabs.Tab>

  </Tabs>
</JsonOnly>
